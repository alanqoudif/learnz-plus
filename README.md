# تطبيق الحضور والغياب للمعلمين

تطبيق بسيط وسهل الاستخدام لإدارة حضور وغياب الطلاب، مصمم خصيصاً للمعلمين.

## المميزات

- **تسجيل دخول بسيط**: رقم الهاتف والاسم فقط
- **إدارة الفصول**: إضافة فصول دراسية وشعب متعددة
- **إدارة الطلاب**: إضافة الطلاب يدوياً
- **تسجيل الحضور**: واجهة سهلة مع إيماءات السحب
- **حفظ البيانات**: حفظ جميع البيانات محلياً على الجهاز
- **واجهة عربية**: تصميم باللغة العربية مع دعم RTL

## كيفية الاستخدام

### 1. تسجيل الدخول
- أدخل اسمك ورقم هاتفك
- اضغط على "تسجيل الدخول"

### 2. إضافة فصل دراسي
- اضغط على "إضافة فصل" في الصفحة الرئيسية
- أدخل اسم الفصل (مثل: الخامس، السادس)
- أدخل الشعبة (مثل: أ، ب، ج)
- اضغط على "إضافة الفصل"

### 3. إدارة الطلاب
- اضغط على "إدارة الطلاب" في الفصل المطلوب
- اضغط على "إضافة طالب"
- أدخل اسم الطالب
- كرر العملية لكل طالب

### 4. تسجيل الحضور
- اضغط على "تسجيل الحضور" في الفصل المطلوب
- اضغط على "بدء التسجيل"
- اسحب كارت الطالب:
  - **لليمين** = حاضر
  - **ليسار** = غائب
- أو استخدم الأزرار اليدوية

## التقنيات المستخدمة

- **Expo 54**: منصة تطوير التطبيقات
- **React Native**: إطار عمل التطبيقات المحمولة
- **TypeScript**: لغة البرمجة
- **React Navigation**: للتنقل بين الشاشات
- **AsyncStorage**: لحفظ البيانات محلياً
- **React Native Gesture Handler**: لإيماءات السحب

## التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- Expo CLI
- تطبيق Expo Go على الهاتف

### خطوات التثبيت

1. **تثبيت المكتبات**
   ```bash
   npm install
   ```

2. **تشغيل التطبيق**
   ```bash
   npm start
   ```

3. **فتح التطبيق**
   - افتح تطبيق Expo Go على هاتفك
   - امسح QR Code الذي يظهر في المتصفح
   - أو اضغط على "Run on iOS simulator" أو "Run on Android device/emulator"

### أوامر إضافية
```bash
# تشغيل على Android
npm run android

# تشغيل على iOS
npm run ios

# تشغيل على الويب
npm run web
```
